35.5.DefiningCommonContentUsingLayouts

ActiveClass This attribute specifies one or more CSS classes that the anchor element rendered by the NavLink component
will be added to when the current URL matches the href attribute value.
Match This attribute specifies how the current URL is matched to the href attribute, using a value from the
NavLinkMatch enum. The values are Prefix, which considers a match if the href matches the start of the URL,
and All, which requires the entire URL to be the same.

InvokeAsync<T>(name, args) This method invokes the specified function with the arguments provided. The result type is
specified by the generic type parameter.
InvokeVoidAsync(name, args) This method invokes a function that doesn’t produce a result.
