How to cache data in .NET core?
In Asp.Net Core support some way to help cache data such as In-memory cache, Distributed cache and Response cache using middleware. We can cache by URL, cache by method, function name... It depends on your purpose if you want to cache data only you can choose In-memory cache and if you have some web API and want to cache it you can choose Response cache middleware.
